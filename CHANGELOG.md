# Changelog

## [0.4.0] - 2026-06-23

### Added

- `package_stack_interstitial_3mf.py` for packaging an aligned stack STL and
  interstitial STL into one import-friendly 3MF.
- Unit test coverage for generated 3MF package structure.

## [0.3.0] - 2026-06-15

### Added

- Interface quality reports for separator candidate planes.
- Unit tests for broad, tiny, fragmented, and missing interface classifications.

## [0.2.0] - 2026-06-15

### Added

- Bed-area savings estimates comparing flat XY cloning against vertical stacks.
- Unit tests for bed batch savings, rotated bed fit, and oversized part errors.

## [0.1.0] - 2026-06-15

### Added

- Initial stackability scoring CLI for binary STL files.
- Unit tests for broad flat and tall narrow stackability candidates.
