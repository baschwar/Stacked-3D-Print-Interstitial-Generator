# Examples

This folder includes both a tiny generated demo and full-size real-world STL
examples.

Run `make_demo_frame.py` to create a tiny binary STL frame, then run the stack
and interstitial scripts against it. The generated STL outputs are ignored by
Git so the repository stays lightweight.

The `gridfinity-baseplates/` folder contains:

- `before/`: already-stacked 8-up Gridfinity baseplate STLs.
- `after/`: matching generated interstitial separator STLs.
